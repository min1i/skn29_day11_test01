| 변수 | 의미 | 설명 |
| :--- | :--- | :--- |
| $G_t$ | 기울기 누적값 | 현재 단계까지 업데이트된 기울기 제곱의 가중 평균값 |
| $G_{t-1}$ | 이전 누적값 | 직전 단계까지 쌓여있던 기울기 히스토리 |
| $\rho$ (Rho) | 지수 이동 평균 계수 | 과거 기울기와 현재 기울기의 반영 비율을 결정 (보통 0.9) |
| $\eta$ | 학습률 | 전체적인 업데이트 보폭을 결정하는 하이퍼파라미터 |
| $\nabla_{\theta}J(\theta)$ | 현재 기울기 | 현재 위치에서 손실 함수가 가장 가파르게 변하는 방향과 크기 |
| $\epsilon$ (Epsilon)| 안전장치 | 분모가 0이 되어 연산 오류가 발생하는 것을 막는 아주 작은 값 |
| $\theta$| 매개변수 | 업데이트하고자 하는 가중치(Weight)나 편향(Bias) |
