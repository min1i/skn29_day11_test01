# 머신러닝 수업 전체 내용 정리 (Day 19 ~ Day 33)

## 전체 흐름

```
지도학습(회귀/분류) → 비지도학습(군집화/차원축소) → 신경망 기초 → 딥러닝(PyTorch)
```

---

## 1주차: 지도학습 기초 (Day 19~21)

### Day 19 — 선형 회귀 입문
- `y = wx + b` 선형 관계 모델링
- 전처리: `StandardScaler` (평균 0, 표준편차 1)
- 평가지표: MAE, MSE, R²
- 데이터셋: Diabetes

### Day 20 — 회귀 심화 & 파이프라인 (5개 노트북)
- **계층화 샘플링**: `pd.cut()` + `stratify`
- **전처리 파이프라인**: `ColumnTransformer` + `OneHotEncoder`
- **이상치 탐지**: `IsolationForest`
- **로그 변환**: `np.log1p()` — 왜도 큰 피처 정규화
- **교차검증**: `KFold`, `StratifiedKFold`, `cross_val_score`
- **모델 비교**: Linear < DecisionTree < RandomForest (R² 기준)
- `GridSearchCV` 하이퍼파라미터 탐색
- 데이터셋: Housing (20,640개), Diamonds

### Day 21 — 의사결정나무 & 앙상블
- `DecisionTreeClassifier`: 지니 불순도 기반 분할
- `feature_importances_`: 특성 중요도
- `plot_tree()`: 트리 시각화
- **PCA**: 4차원 → 2차원 축소
- **앙상블 4종류**:
  - Hard Voting (다수결)
  - Soft Voting (확률 평균)
  - Bagging (중복 표집)
  - Stacking (메타모델)

## 2주차: 분류 심화 (Day 22~24)

### Day 22 — SGD & 다중 분류
- Sigmoid 함수, 경사하강법(SGD)
- `Confusion Matrix`, `Classification Report`
- 다중 분류 (10클래스), `class_weight='balanced'` (불균형 처리)
- ROC-AUC 곡선
- 데이터셋: MNIST (손글씨 0~9)

### Day 23 — SVM & 앙상블 + LightGBM
- **SVM**: `LinearSVC`, `SVC(kernel='rbf', 'poly')`
- `decision_function()` 결정 경계 시각화
- `VotingClassifier` (Hard/Soft)
- **LightGBM** 첫 등장: `RandomizedSearchCV`로 튜닝
- 성능: LR < DT < RF < LightGBM (MAE 기준)

### Day 24 — 군집화 & 지리 데이터
- **DBSCAN**: `eps`, `min_samples` 파라미터
- **계층적 군집화**: Single/Complete/Average Linkage, 덴드로그램
- **실전 프로젝트**: 상권 데이터 17개 CSV 통합
- `BallTree`: 반경 500m 내 점포 수 계산
- LightGBM 기반 창업 위치 Top 10 추천

## 3주차: 비지도학습 (Day 25~27)

### Day 25 — 나이브 베이즈, KNN, K-Means
- **Naive Bayes**: 베이즈 정리 `P(C|X) = P(X|C)×P(C) / P(X)`
- **KNN**: 유클리드 거리 기반, 최적 K 교차검증으로 탐색
- **K-Means**: Elbow Method (WCSS/Inertia), Silhouette Score
- PCA(95% 분산 유지) → K-Means 조합

### Day 26 — 군집화 심화 & PCA 실전
- DBSCAN 노이즈 포인트 (-1 라벨)
- **Olivetti Faces 데이터셋**: 400개 얼굴 이미지(64×64)
- PCA(99% 분산): 4096 → 177 차원 축소
- K-Means로 동일 인물 군집화

### Day 27 — 모델 비교 종합 + 퍼셉트론 개념 도입
- 퍼셉트론: AND 성공, **XOR 실패** (단층의 한계)
- SVM vs Decision Tree (Iris 이진분류)
- PCA + Stacking 조합 (Wine 정확도 99%)
- GridSearchCV: DT / RF / LightGBM 파라미터 탐색

## 4주차: 신경망 기초 (Day 28~30)

### Day 28 — 퍼셉트론 & MLP 증명
- **단층 퍼셉트론**: AND 게이트 완벽 분류, XOR 불가
- **활성화 함수 없으면** 100층도 선형 → 의미 없음
- **Sigmoid 문제**: 기울기 최대 0.25 → 소실 위험
- **ReLU 장점**: Sigmoid 대비 24배 기울기 유지
- **MLP로 XOR 해결**: 은닉층 추가로 완벽 분류
- MNIST: 784 → 256 → 10 구조, SGD 학습

### Day 29 — 손실함수 & AutoML
- **손실함수**:
  - MSE: 회귀
  - BCE (Binary Cross Entropy): 이진분류
  - Cross Entropy + Softmax: 다중분류
- 실습: Diabetes(회귀) / Breast Cancer(이진) / Iris(다중)
- **FLAML AutoML**: 자동 모델 탐색

### Day 30 — 역전파 & 경사하강법 심화
- 수치 미분: `(f(x+h) - f(x)) / h`
- **Chain Rule**: 다단계 기울기 계산
- 은닉층 역전파 수식 직접 유도
- 타이타닉 생존 예측 DNN (정확도 76%)

## 5주차: PyTorch 딥러닝 (Day 31~33)

### Day 31 — PyTorch 기초
- **Autograd**: `requires_grad=True`, 자동 미분
- 학습 루프: `zero_grad()` → `backward()` → `step()`
- `BCELoss`, `CrossEntropyLoss`
- XOR 문제 은닉층 4개 → 정확도 100%
- **Pipeline**: Dropout 정규화, `DataLoader` 배치 처리

### Day 32 — 심층 신경망 실전
- **Fashion MNIST**: 28×28 이미지, 10클래스 분류
- ReLU + Softmax 출력층
- Cross Entropy Loss
- 최종 정확도: **88%**

### Day 33 — 과적합 방지 & 정규화 기법
- **과적합 원인**: 복잡한 모델 + 적은 데이터
- **Dropout**: `nn.Dropout(p=0.3~0.5)` — 뉴런 무작위 비활성화
- **Batch Normalization**: `nn.BatchNorm1d()` — 각 레이어 출력 정규화
- **Early Stopping**: 검증 손실 개선 없을 시 학습 중단 (`patience` 파라미터)
- **Learning Rate Scheduler**: `ReduceLROnPlateau` — 검증 손실 정체 시 LR 감소
- `model.train()` / `model.eval()` 모드 전환 필수 (Dropout/BN 동작 다름)
